# Netflix Exploratory Data Analysis

## Objective
Perform exploratory data analysis on Netflix Movies and TV Shows dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Content Type Analysis
- Country-wise Analysis
- Genre Analysis
- Release Year Trends
- Duration Analysis
